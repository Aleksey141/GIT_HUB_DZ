фатальная ошибка: текущая ветка newbranch не имеет вышестоящей ветки. Чтобы отправить текущую ветку и установить вышестоящую ветку, используйте git push --set-upstream origin newbranch Чтобы это происходило автоматически для веток без вышестоящей ветки, см. 'push.autoSetupRemote' в 'git help config'.
